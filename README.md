# enplus-images

本项目主要为[EventerNote Plus](https://github.com/738NGX/eventernote-plus/)（简称EN+）提供原网站没有的图片服务。任何使用者可以提交PR来上传需要的图片资源。具体的对应操作参阅下文。

## 艺人头像

如果您想为EN+贡献艺人头像，但此前从未接触过 GitHub，请按照以下步骤操作。整个过程完全在浏览器中完成，无需安装任何软件。

### 第一步：准备图片

在开始之前，请确保您的图片符合以下要求：

1. **比例**：1:1 正方形（如 300x300 像素）。
2. **格式**：`.webp` 格式。
3. **大小**：单张图片不超过 **100KB**。
4. **命名**：Eventernote的艺人ID（例如 **水樹奈々** 的en主页链接为 `https://www.eventernote.com/actors/水樹奈々/28`，其id为url最后的数字 **28**，请把头像文件命名为 `28.webp` ）。

## 第二步：操作流程

### 1. 登录并复制仓库（Fork）

1. 登录您的 GitHub 账号。
2. 在本页面右上角，找到并点击 **Fork** 按钮。
3. 在弹出的页面中点击 **Create fork**。这会在您的账号下创建一个本项目的副本。

### 2. 进入目标文件夹

1. 在您自己的仓库页面中，点击进入 **`actors`** 文件夹。

### 3. 上传图片

1. 点击右上角的 **Add file** 按钮，选择 **Upload files**。
2. 将准备好的 `.webp` 图片拖入上传区域，或者点击选择文件。
3. 等待上传进度条完成。

### 4. 提交更改（Commit）

1. 在下方的 **Commit changes** 框中，输入简短的描述（例如：“添加 花泽香菜 的头像”）。
2. 确保选中 "Commit directly to the `main` branch"。
3. 点击 **Commit changes** 按钮。

### 5. 发起合并请求（Pull Request）

这是最关键的一步，通知管理员把你上传的图片合并到主项目中：

1. 回到您仓库的主页，你会看到一行黄色的提示条，显示 "This branch is 1 commit ahead..."。
2. 点击右侧的 **Contribute**，然后点击 **Open pull request**。
3. 在新页面中，点击绿色的 **Create pull request** 按钮。
4. 再次点击 **Create pull request** 确认发送。

### 常见问题

* **如何转换 webp 格式？** 您可以使用在线转换工具（如 Squoosh 或 CloudConvert）。
* **图片比例不对怎么办？** 请先使用手机或电脑自带的相册编辑功能裁剪成正方形。
* **我提交错了怎么办？** 您可以关闭该 PR 重新开始。

> **想要确认是否成功？**
> 提交完成后，请观察本项目顶部的 **Pull requests** 标签页。如果你能看到自己刚才命名的标题，说明操作成功！感谢您的贡献。
